## 2. 빌더 실행 옵션

### 2.1 `builder-config.json` 옵션

* `workDir`: 작업 디렉터리
* `outDir`: 결과 출력 디렉터리
* CSS 빌드 설정
  * `baseCss`: 기본 SCSS 파일 위치
  * `priority`: SCSS 병합 순서

### 2.2 명령줄 인수

* `build`: `workDir`을 빌드하여 `outDir`에 출력합니다.
* `export`: `workDir`을 상응되는 하나의 HTML 파일로 출력합니다.
* `import`: HTML 파일을 `dev`로 불러옵니다.
